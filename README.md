La verdad, esta pequeña app la hice para mi partida de rol de Anima. Si alguien quiere utilizarlo, libre es.
Algunos detalles de la misma:
  -Dale a Ordenar para organizar los valores de mayor a menor iniciativa total.
  -Inicia el combate y ve pasando de personaje con "Siguiente turno"
  -Los personajes se ordenaran segun su nombre despues de finalizar la ronda de combate. Puedes volver a ordenarlos con el
  valor anterior si lo necesitas. El orden siempre será PLATA, TAKESHI, MARAVI, MARTINA (Si estos nombres existen), 
  nombres ordenados por números y finalmente otros que no cumplan las dos condiciones anteriores.
